Named Routes

-Instead of using static routes in blade / front, can use named routes
-helps if change the uri and its not hardcoded route on front end

Use 
->name( )

RoutesRoute::get('/blog', [PostsController::class, 'index'])->name('blog.index');

Static
<a href="/blog">Blog</a>

Dynamic
<a href={{ route('blog.index') }}>Blog</a>

With Params
Route::get('/blog/{id}', [PostsController::class, 'show'])->name('blog.show');

<a href={{ route('blog.show', ['id' => 1]) }}>Blog</a>

Resource Route
Route::resource('blog', PostsController::class);
<a href={{ route('blog.show', ['id' => 1]) }}>Blog</a>
   -will still work
   -names will be automatically binded
