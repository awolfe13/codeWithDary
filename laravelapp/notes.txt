-tables = Migrations
-create tables inside db without using db interface tool
-4 default migrations are related to authentication 
-when migrating, will migrate by date

Migration Names
time stamp_action_name

Artisan
php artisan make:migration create_posts_table

(2 opt params)
php artisan make:migration create_posts_table --create=posts
  -prefill migration table named
php artisan make:migration create_posts_table --table=posts
  -modification to existing table

Inside Migration 
  $table->string('title')->unique();
    -title must be unique

  $table->text('exerpt')->nullable();
    -column can have null values
     
  $table->integer('minutes_to_read')->default(1);
    -gives a default of 1 minutes

  $table->string('image_path');   
     -cant store image, so will store the path to the image

 Migrate the Tables to DB    
 php artisan migrate
    -runs all up methods that havent been run

Show all Migrations
php artisan migrate:status

Rollback All Migrations
php artisan migrate:reset

Rollback And Migrates 
php artisan migrate:refresh
